Avoid Lock In

If you decide that you don't want your data stored on SimpleText.ws, it's easy to move on. The site’s code is open source, you can host it on your own server, and there’s a public API for getting your data out.

Open Source

The server code is open source. The Google Web Toolkit based client code is also open source. Please take a look. If you notice something that I could do better let me know and I'll incorporate your change into the next release.

Self Hosting

You can host the code using any server that conforms to the Google App Engine environment. Server options include:

  • Use your own Google App Engine account following these instructions.
  • Use Google App Engine’s SDK to host from your computer.
  • Use AppScale, the open source implementation of Google App Engine.

Public API

SimpleText.ws has a public API so that other applications can sync with your data. The public API uses Google for authentication, JSON for data formatting, and has a RESTful design. The API is demonstrated and documented by this python client library.
